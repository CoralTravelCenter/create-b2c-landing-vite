# create-b2c-landing-vite

Креатор проектов для `b2c-landing-vite`. Это рекомендуемый и единственный штатный способ создать новый проект лендинга.

## Использование

Требуются Node.js 18 или новее и npm.

После публикации пакета выполните:

```bash
npm create b2c-landing-vite
```

Креатор:

1. запросит имя проекта;
2. создаст одноимённую папку из встроенного шаблона;
3. запишет имя проекта в `package.json`;
4. установит зависимости;
5. выведет команды для запуска.

После создания:

```bash
cd <project-name>
npm run dev
```

Папка с таким именем не должна существовать или должна быть пустой.

Шаблон содержит стартовый блок `welcome`, изображения в `public/` и npm-скрипты, которые вызывают CLI пакета `b2c-landing-vite`. Подробное описание структуры проекта, блоков, Tampermonkey и CMS-сборки находится в README билдера.
